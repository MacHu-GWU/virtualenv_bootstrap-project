Install
------------------------------------------------------------------------------

``virtualenv_bootstrap`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install virtualenv-bootstrap

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade virtualenv-bootstrap
